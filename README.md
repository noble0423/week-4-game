# CRYSTALS GAME

## Objective

In this game, your objective is to make your score match the value that the computer is thinking.

### Instructions

* Reveal the computer's hidden value by clicking on the "?" button.

* Each of the four crystals at the bottom of the page contains a hidden value that is randomly chosen each round.

    * Click each crystal in order to add it's hidden value to the score panel.

* Continue to increase your score by clicking more crystals.

* Make your score match the computer's value and you win!

    * Go over and you lose!

* The game is automatically reset after each win or loss.

    * The computer's value and all four crystals will all have new random values assigned to them. 




* Hint: The only way to determine the hidden value of each crystal that you click is to evaluate the change in score.
